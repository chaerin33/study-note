## 🔍 문제별 출제 예상

### 문제 1 — 시스템 모델링 + 안정성 분석

> 기계/전기 시스템이 주어지고, State-Variable Form으로 변환 후 Matrix Exponential 계산 → 고유값으로 안정성 분석

**핵심 흐름:**

$$\dot{\mathbf{x}} = A\mathbf{x} + B\mathbf{u} \quad \Rightarrow \quad \mathbf{x}(t) = e^{At}\mathbf{x}(0)$$

**출제 포인트:**

- **Cayley-Hamilton Theorem**으로 $e^{At}$ 계산
  $$e^{At} = \alpha_0(t)I + \alpha_1(t)A + \cdots + \alpha_{n-1}(t)A^{n-1}$$
- **중복 고유값(Multiplicity)**: 미분 보충법으로 $t^k e^{\lambda_i t}$ 항 생성
- **Eigendecomposition**: $A = U\Lambda U^{-1}$
- **안정성 판별 기준**

| 조건 | 판정 |
|------|------|
| 모든 $\text{Re}(\lambda_i) < 0$ | 점근 안정 (Asymptotically Stable) |
| 모든 $\text{Re}(\lambda_i) \leq 0$ | 한계 안정 (Marginally Stable) |
| $\exists\, \text{Re}(\lambda_i) > 0$ | 불안정 (Unstable) |

- **검증**: $e^{At}\big|_{t=0} = I$ 반드시 확인

**관련 주차**: Week 1, 7, 8

### 문제 2 — Linearization (선형화)

> 비선형 ODE가 주어지고, 평형점 설정 후 Jacobian 선형화 수행

**핵심 흐름:**

$$x(t) = \bar{x} + \hat{x}(t) \quad \Rightarrow \quad \dot{\underline{\xi}} = J_x\big|_{\bar{x},\bar{u}} \cdot \underline{\xi} + J_u\big|_{\bar{x},\bar{u}} \cdot \delta\underline{u}$$

**출제 포인트:**

- **평형점(Operating Point) $\bar{x}$ 설정**: $\dot{x} = 0$ 조건으로 결정
- **Taylor 1차 전개**
  $$f(x) \approx f(\bar{x}) + \left.\frac{df}{dx}\right|_{\bar{x}} \hat{x}$$
- **다변수 시스템**: Jacobian 행렬 편미분 적용
  $$J_x = \frac{\partial \underline{f}}{\partial \underline{x}}\bigg|_{\bar{x},\bar{u}}$$
- **중력항 소거**: $\bar{x} = mg/k$ 대입 후 $mg$와 $k\bar{x}$ 상쇄 확인
- 선형화 후 **시스템 행렬 $A$** 구성까지 완성

**관련 주차**: Week 4, 5

### 문제 3 — Fourier Transform

> 시간 신호 $x(t)$가 주어지고 FT 계산 및 스펙트럼 해석

**핵심 흐름:**

$$\hat{X}(\omega) = \mathcal{F}\{x(t)\} = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^{\infty} x(t)\, e^{-i\omega t}\, dt$$

**출제 포인트:**

- **코사인/사인 FT 결과**

| 신호 | FT 결과 | 크기 | 위상 |
|------|---------|------|------|
| $A\cos(\omega_0 t)$ | $A[\delta(\omega-\omega_0)+\delta(\omega+\omega_0)]$ | $A$ | $0$ |
| $A\sin(\omega_0 t)$ | (허수축 임펄스) | $A$ | $-\pi/2$ |

- **DFT 파라미터**

| 기호 | 의미 | 식 |
|------|------|-----|
| $\Delta t$ | 샘플링 간격 | $T/N$ |
| $\Delta f$ | 주파수 해상도 | $1/T$ |
| $f_s$ | 샘플링 주파수 | $N/T$ |
| $f_H$ | Nyquist 주파수 | $f_s/2$ |

- **Aliasing**: $f > f_H$ 영역에서 가짜 신호 발생

**관련 주차**: Week 6
