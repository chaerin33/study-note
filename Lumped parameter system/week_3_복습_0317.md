## 1. 전기자 제어 DC 모터 전달함수 유도

### 기본 회로 방정식 (필기 이미지 1 참조)

모터 시스템은 전기계(armature 회로)와 기계계(회전체)의 두 부분으로 구성된다.

**전기계 방정식 (Kirchhoff's Voltage Law):**
$$e_i(t) = e_R + e_L + E_b = i_a R_a + L_a \frac{di_a}{dt} + K_V \omega(t) \quad \cdots \text{①}$$

**기계계 방정식 (Newton's 2nd Law for rotation):**
$$J\dot{\omega}(t) + B\omega(t) = T_m = K_T i_a(t) \quad \cdots \text{②}$$

여기서 $K_T \approx K_V \equiv K$ (토크 상수 = 역기전력 상수, 측정을 통해 확인 가능).

**유도 과정 요약:**
- 출발: $e_i = i_a R_a + L_a \dot{i}_a + K_V \omega$, $J\dot{\omega} + B\omega = K_T i_a$
- 변환: 영 초기조건(Zero ICs)에서 $D \rightarrow s$ (Laplace 변환)
- ③: $(L_a s + R_a) I_a(s) + K_V W(s) = E_i(s)$
- ④: $(Js + B) W(s) = K_T I_a(s)$

### 전달함수 유도

**$I_a(s)/E_i(s)$ 구하기** — $W(s)$ 소거:

식 ④에서 $W(s) = \frac{K_T}{Js+B} I_a(s)$를 식 ③에 대입하면:

$$\left[(L_a s + R_a) + \frac{K_T K_V}{Js+B}\right] I_a(s) = E_i(s)$$

$$\Rightarrow \frac{I_a(s)}{E_i(s)} = \frac{Js+B}{(L_a s + R_a)(Js+B) + K_T K_V} \quad \cdots \text{⑥}$$

**$W(s)/E_i(s)$ 구하기** — $I_a(s)$ 소거:

식 ④에서 $I_a(s) = \frac{Js+B}{K_T} W(s)$를 식 ③에 대입하면:

$$\left[\frac{(L_a s + R_a)(Js+B)}{K_T} + K_V\right] W(s) = E_i(s)$$

$$\Rightarrow \frac{W(s)}{E_i(s)} = \frac{K_T}{(L_a s + R_a)(Js+B) + K_T K_V} \quad \cdots \text{⑦}$$

### $L_a \approx 0$ 근사 (단순화)

$L_a$는 모터 내부 armature의 인덕턴스로, 실제로 매우 작다. $L_a = 0$ 적용 시:

$$\frac{I_a(s)}{E_i(s)} \approx \frac{Js+B}{R_a(Js+B) + K_T K_V} = \frac{\frac{J}{B_1}s + \frac{1}{B_1}}{\tau s + 1} = \frac{G_1 s + G_2}{\tau s + 1} \quad \cdots \text{⑧}$$

$$\frac{W(s)}{E_i(s)} \approx \frac{K_T}{R_a(Js + B) + K_T K_V} = \frac{K_T/B_1}{\tau s + 1} = \frac{G_3}{\tau s + 1} \quad \cdots \text{⑨}$$

여기서 $B_1 = R_a B + K_T K_V$, $\tau = \frac{R_a J}{B_1}$, $G_1 = \frac{J}{B_1}$, $G_2 = \frac{B}{B_1}$, $G_3 = \frac{K_T}{B_1}$.

> 💬 **시스템 차수** (질문)
> 식 ⑧, ⑨에서 분모의 최고차항이 $s^1$ → **1차 시스템(first-order system)**. $L_a \neq 0$이면 2차 시스템이 되지만, $L_a \approx 0$ 근사로 1차로 취급.

### Pole과 Zero의 물리적 의미

> 💬 **Pole의 물리적 의미** (질문)
> - **Pole**: 전달함수 분모를 0으로 만드는 $s$ 값 → 동차(homogeneous) ODE의 해에서 지수함수의 지수($e^{s_i t}$)에 해당. 자유응답(natural response)의 **자연 주파수(natural frequency)** 또는 **감쇠율** 정보 포함. 시스템 안정성 결정.
> - **Zero**: 분자를 0으로 만드는 $s$ 값 → 입력 조건의 동적 특성에서 비롯. 해당 값에서 입력 자체를 소거하는 효과. 식 ⑧의 분자 $G_1 s + G_2$는 입력이 $\dot{i}_a$와 $i_a$ 두 성분의 조합임을 의미.

### 시간응답 (단위 계단 입력 $E_i(t) = u(t)$, $E_i(s) = 1/s$)

**$I_a(s)$:**
$$I_a(s) = \frac{G_1 s + G_2}{\tau s + 1} \cdot \frac{1}{s} = \frac{G_2}{s} + \frac{\tau G_2 - G_1}{\tau s + 1} \quad \text{(부분분수 분해)}$$

$$\Rightarrow \boxed{i_a(t) = G_2 + C\, e^{-t/\tau}} \quad (C = G_2 - G_1/\tau)$$

**$W(s)$:**
$$W(s) = \frac{G_3}{\tau s + 1} \cdot \frac{1}{s} = \frac{G_3}{s} - \frac{\tau G_3}{\tau s + 1}$$

$$\Rightarrow \boxed{\omega(t) = G_3\!\left(1 - e^{-t/\tau}\right)}$$

**물리적 해석:**
- $i_a(t)$: $t=0$에서 초기값 $G_2 + C$로 시작, 지수적으로 최종값 $G_2$에 수렴. (전류가 먼저 흘러 모터를 구동)
- $\omega(t)$: $t=0$에서 0으로 시작, 지수적으로 최종값 $G_3$에 수렴. (전류 인가 후 기계계가 반응)

## 2. 앰프(Amplifier) + 모터 시스템

### 시스템 구성 (필기 이미지 2 참조)

$E_d \rightarrow$ **AMP(이득 $G$)** $\rightarrow$ **Motor($R_m$)** $\rightarrow$ 회전체 $(J, B) \rightarrow T_L$

$$T_m = k I_m, \quad E_m = k \omega_m \quad (K_T = K_V \equiv k)$$

**기본 방정식:**
- ①: $G E_d = R_m I_m + E_m = R_m I_m + k\dot{\theta}_m(t)$
- ②: $J\ddot{\theta}_m + B\dot{\theta}_m = T_m - T_L = k I_m - T_L$

$I_m$을 제거하기 위해 ①에서:

$$I_m = \frac{G E_d - k\dot{\theta}_m}{R_m} \quad \cdots \text{③}$$

③을 ②에 대입:

$$J\ddot{\theta}_m + B\dot{\theta}_m = \frac{k}{R_m}\left[G E_d - k\dot{\theta}_m\right] - T_L$$

$$\boxed{J\ddot{\theta}_m + \left(B + \frac{k^2}{R_m}\right)\dot{\theta}_m + T_L = \frac{kG}{R_m} E_d(t)}$$

**왜 이 식인가?** $k^2/R_m$ 항은 Back-EMF에 의한 추가 감쇠 효과. 앰프 이득 $G$가 클수록 입력 $E_d$의 영향이 커진다.

## 3. 유압 시스템 (Hydraulic System)

### 펌프와 모터 분류 (필기 이미지 2 참조)

| 구분 | Centrifugal Pump (CP) | Positive Displacement Pump (PDP) |
|---|---|---|
| 원리 | 내외부 압력차 (원심력) | 고정 체적 변위, 일정 압력/유량 |
| 대표 예 | 터빈 펌프 | 기어 펌프 |

**유압 모터**: 유체 흐름을 받아 회전 운동을 출력 (발전소 터빈 등).

### 동력 유추 (Power Analogy)

세 가지 동력 계산법이 모두 등가:

$$P = F \cdot v \quad \text{(병진계)} = T \cdot \omega \quad \text{(회전계)} = \Delta P \cdot Q \quad \text{(유체계)}$$

단위 확인: $\frac{N}{m^2} \cdot \frac{m^3}{s} = \frac{N \cdot m}{s}$ ✓

### 유압 모터/펌프 기본 관계식

**이상적 조건 ($P_{in} = P_{out}$):**

**유압 모터:** $\Delta P_m Q_m = T_m \omega_m$

$$\Rightarrow \boxed{T_m = \Delta P_m \cdot D_m}, \quad D_m = \frac{Q_m}{\omega_m}$$

($D_m$: 단위 라디안당 유량, 유량 변위(displacement of flow))

**유압 펌프:** $T_p \omega_p = \Delta P_p Q_p$

$$\Rightarrow \boxed{T_p = \Delta P_p \cdot D_p}, \quad D_p = \frac{Q_p}{\omega_p}$$

### 유압-기계 복합 시스템 방정식 유도 (필기 이미지 2 참조)

**시스템 구성:** $Q_i(t) \rightarrow$ 저장조(용량 $C_f$) $\rightarrow$ 유압 모터($D_m$) $\rightarrow$ 회전체($J, B$)
- 누설(leakage): 저항 $R_f$

**유체 구성요소 유추:**

| 유체계 | 전기계 유추 |
|---|---|
| $Q = C_f \dot{P}$ (저장조) | $i = C \dot{V}$ (커패시터) |
| $\Delta P = R_f Q$ (누설 저항) | $V = Ri$ (저항) |
| $\Delta P = I_f \dot{Q}$ (관성) | $V = L \dot{i}$ (인덕터) |

**방정식 유도:**

유량 연속 방정식:
$$Q_i(t) - Q_{load} = C_f \dot{P}(t)$$
$$Q_{load} = Q_{leak} + Q_{turbine} = \frac{1}{R_f}P + D_m \dot{\theta}_m$$

$$\boxed{C_f \dot{P}(t) + \frac{1}{R_f}P(t) + D_m \dot{\theta}_m(t) = Q_i(t)} \quad \cdots \text{①}$$

기계계 방정식:
$$T_m - B\dot{\theta}_m = J\ddot{\theta}_m$$
$$\boxed{J\ddot{\theta}_m(t) + B\dot{\theta}_m(t) = D_m \cdot P(t)} \quad \cdots \text{②}$$

**두 방정식 의미:**
- ①: 유체 입력 유량 = 저장조 충전 + 누설 + 터빈 소모
- ②: 유압 토크 - 점성 저항 토크 = 관성 × 각가속도

> 💬 **유압 시스템과 전기 시스템의 유추** (질문)
> 저장조 $C_f \leftrightarrow$ 전기 커패시터 $C$, 누설 저항 $R_f \leftrightarrow$ 전기 저항 $R$, 압력 $P \leftrightarrow$ 전압 $V$, 유량 $Q \leftrightarrow$ 전류 $i$. 동일한 방정식 구조를 공유.
