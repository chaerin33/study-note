## 이번 주 한눈에 보기

이번 강의는 두 가지 큰 흐름으로 구성된다. 첫째는 지금까지 다룬 LTI 시스템(state-variable form, 선형화)의 해를 구하기 위해 **Fourier Transform(FT)**이 왜 필요한지 동기 부여 — 시스템 응답에 복수의 주파수가 혼합될 때 시간 영역만으로는 분석이 불가능하기 때문이다. 둘째는 $x(t) = A\cos(\omega_0 t)$에 대한 FT를 직접 유도하여 결과가 Dirac delta 함수($\delta(\omega \pm \omega_0)$)로 나타남을 수학적으로 검증하였다. 이어서 실제 측정·계산에 필수인 **Discrete Fourier Transform(DFT)**의 핵심 파라미터 — 샘플 수 $N$, 시간 간격 $\Delta t$, 주파수 해상도 $\Delta f$, 샘플링 주파수 $f_s$, Nyquist 주파수 $f_H$, aliasing — 를 순서대로 정의하고 그 물리적 의미를 설명했다.

## 1. 강의 도입부: 지금까지의 흐름 정리 & 이번 주 방향

교수님은 지금까지 전기-기계 복합 시스템(DC 모터, 유압계)의 모델링, state-variable 변환, 비선형 선형화를 다뤘음을 상기시키고, **앞으로의 핵심 목표**를 다음과 같이 제시했다.

**Zero input condition 하의 해:**

$$\dot{\underline{x}}(t) = A\,\underline{x}(t) \quad \text{(zero input condition)}$$

이 방정식의 해는 시간의 함수로 표현되며, 가장 일반화된 형태는 미지 상수 벡터와 지수함수의 곱으로 구성된다. 그 해법을 본격적으로 다루기 전에, **Fourier Transform**을 먼저 이해해야 한다고 강조했다.

**왜 주파수 영역이 필요한가?**

실제 시스템 응답은 여러 주파수 성분이 혼합되어 나타난다:

- **선형 시스템:** 입력 주파수와 동일한 하나의 주파수만 응답에 존재
- **비선형 시스템:** 기본(fundamental) 주파수 외에 추가 성분이 결합됨

| 용어 | 의미 |
|------|------|
| **Superharmonic** | 기본 주파수보다 *빠른*(높은) 주파수 성분 |
| **Subharmonic** | 기본 주파수보다 *느린*(낮은) 주파수 성분 |

> 💬 **[superharmonic vs subharmonic]** (질문)
> 비선형 시스템에서 응답에 기본 주파수 외 성분이 섞일 때, 더 빠른 성분은 superharmonic, 더 느린 성분은 subharmonic이라 부른다. 시간 영역에서 이 두 성분이 혼합된 복잡한 파형은 Fourier Transform을 통해 주파수 영역으로 변환해야 각 성분의 크기와 위상을 명확히 구분할 수 있다.

## 2. Fourier Transform (FT) 정의

**기본 정의 쌍:**

$$\hat{X}(\omega) = \mathcal{F}\{x(t)\} = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^{\infty} x(t)\, e^{-i\omega t}\, dt$$

$$x(t) = \mathcal{F}^{-1}\{\hat{X}(\omega)\} = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^{\infty} \hat{X}(\omega)\, e^{i\omega t}\, d\omega$$

**계수 표기 관습에 대한 주의:**

$$\mathcal{F}\{f(t)\} = \frac{1}{2\pi}\int_{-\infty}^{\infty} f(t)\, e^{-i\omega t}\, dt = \hat{f}(\omega)$$

교수님은 $\frac{1}{\sqrt{2\pi}}$ vs. $\frac{1}{2\pi}$ 표기 두 가지가 모두 사용됨을 지적했다. 계수를 순변환에 $\frac{1}{\sqrt{2\pi}}$ 붙이면 역변환에도 $\frac{1}{\sqrt{2\pi}}$, 순변환에 생략하면 역변환에 $\frac{1}{2\pi}$가 붙는다. **핵심은 어느 쪽을 선택하든 일관성을 유지하는 것.**

**FT의 물리적 의미:**

$$x(t) \xrightarrow{\mathcal{F}} \hat{X}(\omega) = |\hat{X}(\omega)| \angle \phi(\omega)$$

- $|\hat{X}(\omega)|$: 각 주파수 성분의 **크기(magnitude)**
- $\phi(\omega)$: 각 주파수 성분의 **위상각(phase)**

## 3. $A\cos(\omega_0 t)$의 Fourier Transform 유도

**출발:**

$$\hat{X}(\omega) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^{\infty} A\cos(\omega_0 t)\, e^{-i\omega t}\, dt$$

**$e^{-i\omega t} = \cos(\omega t) - i\sin(\omega t)$ 대입:**

$$= \frac{A}{\sqrt{2\pi}} \left[ \int_{-\infty}^{\infty} \cos(\omega_0 t)\cos(\omega t)\, dt - i\int_{-\infty}^{\infty} \sin(\omega t)\cos(\omega_0 t)\, dt \right]$$

**적화공식(product-to-sum) 적용:**

$$\cos(\omega_0 t)\cos(\omega t) = \frac{1}{2}\left[\cos((\omega-\omega_0)t) + \cos((\omega+\omega_0)t)\right]$$

$$\sin(\omega t)\cos(\omega_0 t) = \frac{1}{2}\left[\sin((\omega-\omega_0)t) + \sin((\omega+\omega_0)t)\right]$$

**적분 후:**

$$= \frac{A}{2\sqrt{2\pi}} \left[ \frac{\sin((\omega-\omega_0)t)}{\omega-\omega_0} + \frac{\sin((\omega+\omega_0)t)}{\omega+\omega_0} \right]_{-\infty}^{\infty} + i(\cdots)$$

**핵심 계산 결과 ($-\infty$~$\infty$ 적분):**

$$\int_0^T A\cos(\omega t)\, dt = \begin{cases} A\cdot T & \omega = 0 \\ 0 & \omega \neq 0 \end{cases}$$

→ 코사인 함수를 $-\infty$~$\infty$에서 적분하면 $\omega \neq 0$일 때 양과 음이 상쇄되어 0이 된다. 이것이 Dirac delta의 수학적 근거.

> 💬 **[cos 적분 결과]** (질문)
> 코사인 함수를 한 주기 이상 적분할 때 $\omega \neq 0$이면 0이 되는 이유는? 코사인의 양의 반주기와 음의 반주기 넓이가 서로 정확히 상쇄되기 때문이다. $\omega = 0$일 때만 $\cos(0) = 1$이 되어 $A\cdot T$가 남는다.

**최종 결과:**

$$\mathcal{F}\{A\cos(\omega_0 t)\} = A\left[\delta(\omega - \omega_0) + \delta(\omega + \omega_0)\right]$$

여기서 **Dirac delta 함수**:

$$\delta(\omega) = \begin{cases} 1 & \omega = 0 \\ 0 & \omega \neq 0 \end{cases}$$

**물리적 해석:**

- 순수 코사인 신호는 주파수 영역에서 $\omega = \pm\omega_0$에만 크기 $A$의 임펄스가 존재
- **크기(Magnitude):** $|\hat{X}(\omega_0)| = A$
- **위상(Phase):** $\phi(\omega_0) = 0$ (코사인은 위상 = 0)

**사인 함수의 경우:**

$A\sin(\omega_0 t) = A\cos(\omega_0 t - \pi/2)$이므로, 크기는 동일하게 $A$이지만 위상이 $-\pi/2$로 나타난다.

$$\phi(\omega_0) = -\frac{\pi}{2} \quad \text{(sine의 경우)}$$

## 4. Discrete Fourier Transform (DFT) 파라미터 정의

실제 실험 데이터는 연속 신호가 아닌 **유한 개의 이산 데이터**로만 얻어진다. 이를 처리하기 위한 DFT의 핵심 파라미터를 순서대로 정의한다.

### 기본 설정

연속 신호 $x(t)$를 한 주기 $T$ 내에서 $N$개의 데이터로 이산화:

$$t_0, t_1, t_2, \ldots, t_k, \ldots, t_{N-1} \quad \text{(총 } N\text{개, } t_N = T \text{는 제외)}$$

> **왜 $t_N$ 제외?** $t_0$은 1번째 주기의 시작점, $t_N = T$는 2번째 주기의 시작점 → 포함하면 $N+1$개가 되어 중복 계산

주파수 영역에서도 동일하게 $N$개의 데이터:

$$f_0, f_1, f_2, \ldots, f_k, \ldots, f_{N-1}$$

### 핵심 파라미터 (6가지)

**① 시간 간격 $\Delta t$ (= $h$):**

$$\Delta t = t_{k+1} - t_k = h = \frac{T}{N}$$

**② 주파수 해상도 $\Delta f$:**

$$\Delta f = \frac{1}{T}$$

주파수 축에서 인접 데이터 간의 간격. 시간 주기 $T$가 길수록 주파수 해상도가 높아진다.

**③ $k$번째 주파수:**

$$f_k = k \cdot \Delta f = \frac{k}{T} \quad (k = 0, 1, 2, \ldots, N-1)$$

각속도로 표현하면:

$$\omega_k = 2\pi \cdot f_k = \frac{2\pi}{T} \cdot k = \frac{2\pi}{T} \cdot n \quad (n = 0, 1, 2, \ldots)$$

**④ 샘플링 주파수 $f_s$:**

$$f_s = N \cdot \Delta f = \frac{N}{T}$$

$f_s$가 높다는 것 = 같은 시간 $T$ 안에 더 많은 데이터($N$)를 취득한다는 의미.

**⑤ Nyquist 주파수 $f_H$:**

$$f_H = \frac{f_s}{2} = \frac{N}{2T}$$

분석 가능한 **최대 주파수의 이론적 한계**. 이 이상의 주파수 정보는 **aliasing** 발생.

**⑥ 실용 주파수 $f_N$ (Practical Frequency):**

$$f_N \leq f_H = \frac{f_s}{2}$$

산업 현장에서는 aliasing 여유를 두어 실제로는 $0.2 \sim 0.4 \times f_s$로 보수적으로 설정.

> 💬 **[샘플링 주파수의 의미]** (질문)
> 샘플링 주파수가 높으면 어떤 의미인가? → 자본 $N$이 증가 = 같은 $T$ 안에 더 촘촘하게 측정. 그러나 분석 목표 주파수 대역의 2배 이상이면 충분 — 그 이상 늘리는 것은 효율적이지 않다.

## 5. Aliasing (앨리어싱)

**Aliasing의 의미:**

Nyquist 주파수 $f_H$를 초과하는 주파수 영역의 DFT 결과는 $0 \sim f_H$ 영역의 **거울 상(mirror image)**으로 나타난다. 이는 실제로 존재하지 않는 **가짜 데이터(fake data)**이므로 분석에서 제거해야 한다.

$$\text{Aliasing region: } f > f_H \Rightarrow \text{mirror image of } f < f_H$$

**DFT 전체 결과 요약 (코사인 신호):**

| 주파수 위치 | 크기 | 위상 |
|------------|------|------|
| $f = f_1 = \omega_0/(2\pi)$ | $A$ | $0$ |
| $f = f_{N-1}$ (Aliasing 대칭점) | $A$ | $0$ |
| 그 외 모든 주파수 | $\approx 0$ | 수치 오차 수준 |
| $f > f_H$ (Aliasing 영역) | 거울 상 | 제거 필요 |

> 💬 **[Aliasing 물리 의미]** (질문)
> Aliasing이란? 교수님 설명: 앞에 서면 같은 모습이 비치는 거울처럼, Nyquist 이상 주파수 데이터는 낮은 주파수 영역의 복사본으로 나타난다. 실제 신호가 아니라는 의미에서 "가짜 신호(fake/alias)"라 부른다.

**Aliasing 방지 원칙 (Nyquist 기준):**

분석하고자 하는 최대 주파수가 $f_{max}$일 때, 샘플링 주파수는 반드시:

$$f_s \geq 2 \times f_{max}$$

## 6. 시간 주기 $T$ 선택의 중요성

> 💬 **[T 선택 기준]** (질문)
> 시간 주기 $T$를 어떻게 정해야 하는가?

$T$는 분석하려는 신호에서 **가장 긴 주기(가장 낮은 주파수)**를 포함하는 값으로 설정해야 한다.

- 기본 주파수만 있다면: $T$ = 기본 주기 1 사이클로 충분
- **Subharmonic이 존재하는 경우:** $T$는 subharmonic의 주기에 맞춰야 함 (subharmonic이 가장 긴 주기)
- Subharmonic은 비선형 계에서 발생하며(Week 5의 Bifurcation과 연결), 이를 놓치면 그 성분을 분석할 수 없다.
